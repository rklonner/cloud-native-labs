# Backstage - Basic backstage application deployed in Kubernetes

**Demonstrate**:
* How to init a Backstage application
* How to build a Backstage image and load it into kind cluster
* How to setup Could Native PG for Backstage
* How to configure a Backstage deployment

## Prepare environment

### Kind cluster
```bash
kind create cluster --name backstage
```
